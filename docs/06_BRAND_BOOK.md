# Brand Book

## Brand expression

7ice should feel clear, capable, and reassuring. The brand supports decisions and action; it does not compete with critical operational information. Public and authenticated experiences share a family resemblance while internal tools prioritize scanability and error prevention.

## Voice

Write plainly, use active verbs, and lead with the action or outcome. Explain unfamiliar terms once and link to help when needed. Avoid blame, legalistic filler, and false certainty. For delays or failures, state what happened, what the user can do, and when to expect the next update.

## Visual rules

- Use approved logo assets with clear space and never distort, recolor arbitrarily, or place them on insufficient-contrast surfaces.
- Use color as reinforcement, never as the only status signal. Reserve destructive red for destructive or blocking states.
- Product imagery must respect consent, licensing, and customer confidentiality.
- Tenant customization may modify approved theme tokens and content, not protected interaction or accessibility semantics.

The token system is defined in [Design System](./05_DESIGN_SYSTEM.md). Apply this guidance with [SEO](./26_SEO.md), [Accessibility](./28_ACCESSIBILITY.md), and [CMS](./13_CMS.md).
