# Design System 7ice


## Бренд

Название:
7ice

Сфера:
Климатические системы


## Характер бренда

7ice должен восприниматься как:

- технологичный
- надежный
- современный
- премиальный
- аккуратный


## Визуальное направление

Избегаем:

- стандартных снежинок
- изображений вентиляторов
- шаблонного дизайна кондиционеров


Используем:

- минимализм
- чистые формы
- технологичность
- ощущение контроля


## Цветовое направление

Основная идея:

- темный графит
- белый
- холодные акценты


## Стиль интерфейса

- много воздуха
- крупная типографика
- понятные кнопки
- аккуратные карточки


## Тон общения

Не:

"Установим кондиционер дешево"

А:

"Создаем комфортный климат с профессиональным сервисом"


## Первый экран сайта

Главная идея:

7ice — технологичный сервис климатических решений от подбора до обслуживания.
