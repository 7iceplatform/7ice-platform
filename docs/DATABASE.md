# База данных 7ice Platform

## Основные сущности системы


# 1. User (Пользователь)

Сотрудники компании и их роли.

Хранит:

- ID пользователя
- Имя
- Телефон
- Email
- Пароль
- Роль
- Статус аккаунта
- Дата создания


---

# 2. Client (Клиент)

Основная информация о заказчике.

Хранит:

- ID клиента
- Имя
- Телефон
- Email
- Тип клиента:
  - частное лицо
  - компания

- История обращений
- Дата создания


---

# 3. Object (Объект)

Место выполнения работ.

Хранит:

- ID объекта
- Клиент
- Адрес
- Тип помещения:
  - квартира
  - дом
  - офис
  - коммерция

- Площадь
- Этаж
- Комментарии
- Фотографии


---

# 4. Lead (Заявка)

Первичное обращение клиента.

Хранит:

- ID заявки
- Источник заявки:
  - сайт
  - телефон
  - реклама
  - мессенджер

- Клиент
- Описание запроса
- Ответственный менеджер
- Статус:

  - новая
  - в работе
  - замер назначен
  - продано
  - отказ


---

# 5. Order (Заказ)

Продажа оборудования и услуг.

Хранит:

- ID заказа
- Клиент
- Объект
- Оборудование
- Стоимость
- Себестоимость
- Прибыль
- Менеджер
- Статус заказа


---

# 6. Installation (Монтаж)

Выполнение монтажных работ.

Хранит:

- ID монтажа
- Заказ
- Бригада
- Дата монтажа
- Чек-лист работ
- Фото до/после
- Комментарии
- Оценка клиента


---

# 7. Equipment (Оборудование)

Установленная техника.

Хранит:

- ID оборудования
- Клиент
- Объект
- Производитель
- Модель
- Серийный номер
- Дата установки
- Гарантия


---

# 8. Service (Обслуживание)

История сервиса.

Хранит:

- ID обслуживания
- Оборудование
- Клиент
- Инженер
- Дата работы
- Выполненные работы
- Стоимость


---

# 9. Payment (Платежи)

Финансовая информация.

Хранит:

- ID платежа
- Заказ
- Сумма
- Тип платежа:
  - предоплата
  - полный расчет

- Дата
- Статус оплаты


---
